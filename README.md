# Marth Requiem Overhaul (MRO)

A terminal load-order mod for **Requiem** (Skyrim SE 1.6.1170, built
against the LoreRim modlist) that rebalances late-game power scaling.
Early and mid game are untouched; every feature gates on endgame
investment. The player and followers ascend — enemies keep their
Requiem lethality.

Built entirely from Linux with no Creation Kit and no xEdit: the plugin
is generated byte-by-byte from Python, Papyrus compiles under Proton
wine, and the native SKSE plugin builds on GitHub Actions.

## Features (v0.6.x)

- **Elemental uncap + absorb** — resistances scale past 100%; above it,
  that element's damage heals you (full absorb at a configurable
  resistance; overheal spills into stamina/magicka)
- **Physical DR as a Mastery Perk** — damage reduction scales past the
  engine's armor cap, gated and ceilinged by your armor mastery; 99% DR
  demands best-in-list gear AND full mastery (target auto-calibrated
  from the load order's actual best heavy set)
- **14-skill Mastery system** (Custom Skills Framework) — each skill
  unlocks at base 100 and trains through real use (landed hits, casts,
  combat time, craft/barter sessions) on vanilla's L^1.95 curve extended
  to skill 300. Weapon +50% dmg, armor +300 AR, magic +50 skill,
  crafting +25%, Speech barter ladder, Smithing raises temper caps
- **Vendor gold doubled dynamically** — the native plugin doubles the 13
  vendor gold leveled lists in memory at load; adapts to any load order
- **QoL** — carry weight +150 (party-wide), arrow recovery 66%, 3-day
  cell reset
- **MCM** — boss readiness assessment, live mastery progress, tuning
  sliders, live status readout (real DR%, per-element absorb)

## Repo layout

```
MRO_GenerateESP.py   generates MRO.esp + SEQ (never hand-edited)
Source/Scripts/      Papyrus sources (+ compile-only stubs)
native/              CommonLibSSE-NG SKSE plugin (CI-built)
MRO-nofomod/         flat install package
MRO-flat/            FOMOD install package
tools/               compile.sh / dump_record.py / audit_esp.py
docs/                START AT docs/INDEX.md
releases/vX.Y.Z/     immutable release archive (never deleted)
```

## Documentation

**`docs/INDEX.md` is the entry point.** The docs are written so any
capable model or person can continue development from them alone:
project playbook, binary-format guide (every record recipe verified
against vanilla bytes), a symptom→fix debugging cookbook, test
procedures, and the native rewrite plan with per-hook open-source
references.

## Requirements

SKSE64, SkyUI 5.2+, Custom Skills Framework, powerofthree's Papyrus
Extender, PapyrusUtil. Built and tested against LoreRim on Skyrim SE
1.6.1170.
